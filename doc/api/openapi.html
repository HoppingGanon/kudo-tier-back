セッションを作成する

セッションを取得するAPI

path Parameters
service
required
string

連携対象のサービス'twitter'または'google'

version
required
number

連携対象のサービスOAuthバージョン1または2

Request Body schema: application/json

セッションの作成に必要な一時セッション等の情報

sessionId
string

OA1, OA2 セッションID

authorizationCode
string

OA2 認証コード

service
string

OA1, OA2 連携サービス

version
string

OA1, OA2 OAuth認証バージョン

state
string

OA2 発行元をチェックするためのstate

oAuthToken
string

OA1 発行済みのトークン

oAuthVerifier
string

OA1 バックエンドで検証するコード

Responses

http://api.kd-tier.hopgn.com/auth/session/{service}/{version}

Content type
application/json

  • "oAuthToken": "string",
  • "oAuthVerifier": "string"
}
